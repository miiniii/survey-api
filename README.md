# Survey API

Spring Boot 기반의 설문 관리 서비스입니다.  
회사별로 다양한 카테고리의 질문을 설정하고, 사용자가 설문에 응답하면 이를 저장하고 통계를 확인할 수 있습니다.

## 주요 기능

- 회사 등록 및 설문 카테고리 관리
- 질문 유형별 설문 생성 (객관식/체크박스/별점/서술형)
- 회원 등록 및 설문 응답 제출
- 설문 결과 저장 및 통계 제공
- 사용자 활동 로그 기록

## 기술 스택

- Java 17
- Spring Boot 3.4.5
- Spring Data JPA
- Spring Security 
- MySQL
- Thymeleaf
- Lombok
- Gradle
- Swagger
